# CSE212-Prototype
This project focuses on the design and implementation of a secure web-based Inventory Management System (IMS) integrated with a comprehensive cyber security framework.
The proposed system will implement core cybersecurity principles such as authentication, Authorisation, Confidentiality, Integrity, and Accountability.
The framework will include secure login mechanisms, Role-Based Access Control (RBAC), password security enforcement, database protection techniques, logging and monitoring of user activities, and backup mechanisms.
The project aims not only to develop a functional inventory system but also to demonstrate how security controls can be systematically integrated into web-based
applications to mitigate real-world cyber threats.
